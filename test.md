# 功能测试

| 项目     | 值               |
| -------- | ---------------- |
| 版本     | v0.1.7           |
| 测试环境 | Windows/ Andriod |

## 环境准备

- Node.js 20+
- 微信 iLink 已登录
- OpenCode 已安装
- 终端运行 `npx wechat-bridge-opencode --agent opencode`

---

## 1. 启动与登录

| 步骤 | 操作                   | 预期结果                                             | 实际结果 | 备注 |
| ---- | ---------------------- | ---------------------------------------------------- | -------- | ---- |
| 1.1  | 首次运行，无登录态     | 终端显示二维码                                       | Pass     |      |
| 1.2  | 扫码登录               | 登录成功，token 保存到 `~/.wechat-bridge-opencode` | Pass     |      |
| 1.3  | 再次运行（已有 token） | 自动登录，不显示二维码                               | Pass     |      |
| 1.4  | `--login` 参数运行   | 强制重新登录                                         | Pass     |      |
